The purpose of this website is to provide a simple, chart visualized data about the current impact of COVID-19 in the Balkan countries.

The data are coming from the NovelCOVID API which utilizes reliable sources like Worldometer's real-time updates about COVID-19 virus.

About logarithmic chart - Since infectious diseases donâ€™t spread in an even, linear fashion, the logarithmic chart proves to be the best and most effective way of interpreting data and showing how the number of cases is growing and declining over time. You can find more useful information about logarithmic charts and how they help explain the spread of COVID-19 here.

Logarithmic chart implemented on this website shows data for the last 30 days and for the 5 countries with the most cases.
